# Lifting Gears Inspection Certificates

## Professional Certificate Management System

A clean, modern web application designed to track and manage lifting gear inspection certificates with ease.

### Features

✨ **Core Features:**
- 📋 Add and manage lifting gear inspection certificates
- 🔍 Search certificates by gear name, certificate number, inspector, or notes
- 🎯 Filter certificates by status (Valid, Pending, Expired)
- 📊 View real-time statistics on certificate status
- ✏️ Edit existing certificates
- 🗑️ Delete certificates with confirmation
- 💾 Local storage persistence - your data stays with you

### User Interface

🎨 **Modern Design:**
- Clean, professional interface with gradient accents
- Responsive design that works on desktop, tablet, and mobile
- Smooth animations and transitions
- Color-coded status badges
- Interactive cards with hover effects

### How to Use

1. **Navigate to the Application**
   - Visit your GitHub Pages URL

2. **Add a Certificate**
   - Click "Add Certificate" in the navigation
   - Fill in the required fields (Gear Name, Certificate Number, Inspection Date, Next Due Date)
   - Optionally add Inspector Name, Status, and Notes
   - Click "Add Certificate" to save

3. **View Certificates**
   - Go to "View All" section
   - See statistics at the top
   - Browse all your certificates

4. **Search and Filter**
   - Use the search bar to find certificates
   - Use filter buttons to show Valid, Pending, or Expired certificates
   - Combine search and filters for precise results

5. **Manage Certificates**
   - Click "Edit" to modify a certificate
   - Click "Delete" to remove a certificate
   - Confirmations protect against accidental deletion

### Technical Details

**Technology Stack:**
- Pure HTML5, CSS3, and Vanilla JavaScript
- Local Storage API for data persistence
- No external dependencies required
- Fully client-side application

**Browser Support:**
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers

### Data Storage

Your certificate data is stored locally in your browser using the Local Storage API. This means:
- ✅ Your data stays on your device
- ✅ No server uploads
- ✅ Works offline
- ⚠️ Clearing browser data will remove stored certificates
